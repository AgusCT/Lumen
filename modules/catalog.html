Module catalog

Task catalog operations.

The catalog is used to give tasks names, and then query them.

Usage:

    local catalog = require 'catalog'

Functions

register (name) Register a name for the current task
waitfor (name, timeout) Find task with a given name.

Functions

register (name)
Register a name for the current task

Parameters:

  • name: a name for the task

Returns:

    true is successful; nil, 'used' if the name is already used by another task.
waitfor (name, timeout)
Find task with a given name. Can wait up to timeout until it appears.

Parameters:

  • name: name of the task
  • timeout: time to wait. nil or negative waits for ever.

Returns:

    the task if successful; on timeout expiration returns nil, 'timeout'.
namefor (taskd)
Find the name of a given task.

Parameters:

  • taskd: task to lookup.

Returns:

    the name if successful; If the task has not been given a name, returns nil.
